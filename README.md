## Aygün Kazımova — Fan-made Portfolio

**Bu repo** məşhur Azərbaycan müğənnisi **Aygün Kazımova** üçün hazırlanmış fan-made portfolio layihəsidir. Burada musiqi, konsert videoları, promo şəkillər və portfolio üçün hazırlanmış veb səhifə nümunələri toplanıb.

## Haqqında
Bu layihə müəllifin (fan) yaratdığı vizual və multimedia materiallardan ibarətdir.Azərbaycanın Xalq artisti Aygün Kazımovanın yaradıcılığına həsr olunmuş fan-made portfolio layihəsidir. 
Bu sayt müğənninin sənət yolu, vizual üslubu və musiqi irsindən ilhamlanaraq həvəskar şəkildə hazırlanmışdır. 
Rəsmi kommersiya məqsədi daşımır.

## Struktur
- `my portfolio.html` — portfolio səhifəsinin HTML faylı
- `portfolio.css` — dizayn üçün stil faylı
- `assets/` — şəkillər və videolar (multimedia)
- digər fayllar — əlavə şəkil/video və media resurslar

## Qeyd (copyright & fan-made)
Bu repo **fan-made** layihədir — rəsmi deyil. Əgər burada yerləşən hər hansı bir media material müəllif hüququna məxsusdursa və sahibinə məxsus iddia varsa, zəhmət olmasa repo sahibi ilə əlaqə saxlayın; material dərhal silinəcək və ya tələb olunan düzəlişlər ediləcək.

## Texniki tələblər və istifadə
Sadə portfolio faylları olduğu üçün brauzerdə birbaşa açmaq kifayətdir:
1. `my portfolio.html` faylını brauzerdə açın.
2. Əgər lokal server işə salmaq istəyirsinizsə, VS Code terminalında:
Sonra `http://localhost:8000/my\ portfolio.html`-a daxil olun.

## Böyük fayllar
Repo-da videolar və böyük şəkillər var — bu fayllar GitHub üçün böyük ölçülü ola bilər. 100MB-dən böyük faylları push etməyə çalışmaq xəta verə bilər. Əgər çoxlu video saxlamaq istəyirsənsə, Git LFS istifadə etməyi və ya videoları ayrı bir hostingdə (YouTube/Cloud) yerləşdirməyi tövsiyə edirəm.
